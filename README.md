# 🌐 KDRC Web (តោះអានសៀវភៅ)

KDRC Web is the frontend application for the Khmer Digital Reading Community platform.  
It provides an interactive and user-friendly interface for readers and writers to discover, read, and share books.

##  Features
- Browse free and premium books
- Read chapters with a clean reading experience
- Write and publish short stories
- Exchange physical books with the community
- User profiles and activity tracking
- Subscription and digital book purchasing UI

## 🛠 Tech Stack
- Vue 3
- TailwindCSS
- Pinia (State Management)
- Vue Router
- Axios

## 🎯 Goal
Deliver a modern, responsive, and Khmer-friendly reading experience that encourages digital reading culture.
